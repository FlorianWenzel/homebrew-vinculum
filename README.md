# homebrew-vinculum

Homebrew tap for [Vinculum](https://github.com/FlorianWenzel/vinculum).

## Install

```bash
brew tap FlorianWenzel/vinculum
brew install vnclm
```

Or in one line:

```bash
brew install FlorianWenzel/vinculum/vnclm
```

## Upgrade

```bash
brew update
brew upgrade vnclm
```

## Formulae

| Name | Description |
|------|-------------|
| [`vnclm`](Formula/vnclm.rb) | CLI for the Vinculum Kubernetes AI-agent platform |
